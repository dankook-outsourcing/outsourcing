#윤수빈님 졸업작품의 웹 프로젝트.

##1. Ami

원하는 인터렉션:
mouseup() :
	- 코딩 : position 이동 (화면 위에서 아래로)
	- 사운드 : play() interact_1.mp3 (페이지가 로딩 되면 바로 소리 나오게)

mousedown() :
	- 코딩 : position 처음으로 이동 (화면 아래서 위로)
	- 사운드 : stop() interact_1.mp3 & play() AMI.mp3

##9. 규건

	- 사운드 : stop() interact_9.mp3 & play() KK.mp3
